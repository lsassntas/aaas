# Family Requests Bot

Discord-бот (Node.js + discord.js v14) для приёма заявок в “семью” через кнопку и Modal.

## Что делает
- В `PANEL_CHANNEL_ID` публикует сообщение с кнопкой **Подача заявки** (если такого сообщения ещё нет среди последних 10 сообщений).
- По кнопке открывает Modal с полями:
  - Никнейм в игре (уровень) + возраст ИРЛ
  - Сколько играете на проекте
  - Были ли в семьях (если да, то каких)
  - Результаты по урону на ДМ 10 минут
- После отправки формы:
  - Создаёт папку заявки: `applications/Заявка-<Ник>-<applicationId>/form.md`
  - Создаёт отдельный Discord-канал в категории `CATEGORY_ID` и публикует туда embed + кнопки модерации
- По кнопке **Принять** / **Отклонить** (только для роли `MODERATOR_ROLE_ID`):
  - Удаляет папку заявки на диске
  - Пишет лог в `LOG_CHANNEL_ID`
  - Канал заявки в Discord НЕ удаляет

## Баллы за войс (опционально)
- Если указать `VOICE_POINTS_PANEL_CHANNEL_ID`, бот публикует панель “Баллы за войс” с кнопками (как на скрине) и начинает начислять баллы за время в войсе.
- Начисление идёт только когда пользователь:
  - находится в голосовом канале
  - **не** в self/server mute
  - **не** в self/server deaf
  - **не** в suppressed (Stage)
  - **не** в AFK-канале (если задан `VOICE_POINTS_AFK_CHANNEL_ID`)
- Важно: “если он ничего не говорит минуту” невозможно отследить, пока бот **не подключён** к голосовым каналам и **не принимает аудио** (Discord не присылает факт речи “просто так”).
- Покупки отвечают **ephemeral** (видит только пользователь). Если задан `VOICE_POINTS_LOG_CHANNEL_ID`, покупка логируется в этот канал.
- Также есть кнопка **“Выдать баллы”**: работает только для ролей из `VOICE_POINTS_GRANT_ROLE_ID` (можно перечислить несколько через запятую).

## Настройка
1. Скопируйте `.env.example` в `.env`
2. Заполните:
   - `DISCORD_TOKEN`
   - `GUILD_ID`
   - `PANEL_CHANNEL_ID` (куда постить кнопку)
   - `MODERATOR_ROLE_ID` (кто может принимать/отклонять)
   - `CATEGORY_ID` (у вас уже: `1293537956912697375`)
   - `LOG_CHANNEL_ID` (у вас уже: `1486721845137768569`)
   - (опционально) `VOICE_POINTS_PANEL_CHANNEL_ID` (куда постить панель баллов)
   - (опционально) `VOICE_POINTS_LOG_CHANNEL_ID` (куда логировать покупки)
   - (опционально) `VOICE_POINTS_AFK_CHANNEL_ID` (AFK voice channel id сервера)
   - (опционально) `VOICE_POINTS_PER_MINUTE` (сколько баллов начислять за минуту, по умолчанию 1)
   - (опционально) `VOICE_POINTS_GRANT_ROLE_ID` (какие роли могут выдавать баллы вручную, через запятую)

## Запуск
```bash
npm install
npm run dev
```
